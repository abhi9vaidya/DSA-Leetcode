# Progress Tracker

Use this file to track problems you solve, their language, and status.

Columns:
- Problem: zero-padded LeetCode ID and short title (e.g., `0001. two-sum`)
- Language: `java`, `python`, `cpp`, etc.
- Status: `todo`, `in-progress`, `done`, `review`
- Notes: short notes, links, or file path to solution
